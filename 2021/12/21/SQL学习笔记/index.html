基础命令(概览)

  • create table 建表 (贴一下数据类型有哪些)
  • alter table 修改表的结构 (感觉有点像声明一样)
  • insert into 向表中增加记录
  • select 用于从数据库中选取数据, 结果被存在一张结果表中, 称为结果集
  • delete 用于删除表中的行
  • update 用于更新表中记录
  • drop table 删表跑路
